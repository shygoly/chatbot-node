## 🚀 快速开始

### 克隆仓库:

```bash
git clone https://github.com/shygoly/chatbot-node.git
cd chatbot-node
```

### 安装依赖:

```bash
npm install
```

### 配置环境:

```bash
cp .env.example .env
# 编辑 .env 填写配置
```

### 初始化数据库:

```bash
npm run db:push
npm run db:seed
```

### 启动服务:

```bash
npm run dev
```
